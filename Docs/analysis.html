1. Introduction

    1.1 Project Summary

    Snapchef is a phone application that are on Android and iOS platforms. The app receives recipes from a database based on various inputs such as pictures of ingredients, UPC barcodes, and text of ingredients, organizing them based on the users preferences.

    1.2 Project Purpose

    The Analysis Document describes the features and platforms for SnapChef and describes how they work.
